# 👗 Student Fashion Wishlist Website

## Overview

The **Student Fashion Wishlist Website** is a front-end web project developed to promote Vietnamese fashion brands among university students. The platform provides users with an engaging space to explore fashion styles, discover local brands, and access outfit inspiration tailored to students' lifestyles and budgets.

By highlighting affordable and high-quality Vietnamese fashion products, the project aims to encourage young consumers to support local businesses while expressing their personal style.

---

## Objectives

* Promote Vietnamese fashion brands to young consumers.
* Help students discover affordable and quality fashion products.
* Encourage the spirit of supporting local brands and products.
* Provide fashion inspiration and styling recommendations for students.

---

## Features

### Brand Discovery

* Browse information about popular Vietnamese fashion brands.
* Learn about each brand's identity, products, and style.

### Fashion Style Exploration

* Explore different fashion styles and trends.
* Discover outfit ideas suitable for students.

### Outfit Collection

* View fashion collections and styling tips.
* Gain inspiration for daily outfits.

### User Account Management

* Register a new account.
* Log in to the system.
* Manage personal profile information.
