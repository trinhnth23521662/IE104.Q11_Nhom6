# IE104.Q11_Nhom6
 
### Tên Website: **Website wishlist thời trang sinh viên** 
   Sinh viên luôn cần thời trang vừa túi tiền, vừa thể hiện được cá tính. Trang web này giúp bạn khám phá những thương hiệu Việt trẻ trung, chất lượng và phù hợp với lối sống năng động – để bạn mặc đẹp mỗi ngày mà vẫn ủng hộ hàng Việt. 
   
### Mục tiêu:
  - Quảng bá thương hiệu thời trang Việt đến giới trẻ  
  - Hỗ trợ sinh viên tiếp cận sản phẩm chất lượng, giá phù hợp  
  - Thúc đẩy tinh thần “người Việt dùng hàng Việt”  
### Công cụ và ngôn ngữ sử dụng
  **HTML, CSS, JavaScript:** Xây dựng cấu trúc, bố cục và hiệu ứng giao diện người dùng
  
  **GitHub:** Lưu trữ và quản lý mã nguồn
### Quản lý thư mục
  **1. css**: Chứa các file css định dạng giao diện cho từng trang
  - File **style.css** định dạng chung cho header và footer
  
  **2. images**: Chứa css file hình ảnh
  
  **3. js**: Chứa các file javascript
  
  **4. README.md**: Mô tả dự án
  
  **5. Các file html**: Các trang web 
  - index.html: Trang chủ giới thiệu thương hiệu thời trang Việt Nam
  - khampha.html: Trang khám phá chung
      + vintage.html: Trang giới thiệu về phong cách Vintage
  - brands.html: Trang giới thiệu về các thương hiệu thời trang Việt Nam
      + levents.html: Trang giới thiệu về Levents
      + dirtycoins.html: Trang giới thiệu về DirtyCoints
  - collection.html: Trang bộ sưu tập giới thiệu một vài tips phối đồ
  - about.html: Trang giới thiệu về dự án và nhóm thực hiện
  - dangky.html: Trang đăng ký tài khoản
  - dangnhap.html: Trang đăng nhập
